MVP Development Plan for Kusina de Amadeo
Core MVP Features
1. Authentication System (Essential)
Google OAuth only
Basic role system (customer/admin)
Required user profile fields:
Email (from Google)
Full name
Phone number
Basic address info
2. Product Management (Core)
Categories (4 main categories only)
Products with:
Basic info (name, description, price)
Single image
Category assignment
Stock status (in stock/out of stock)
Variants system:
Size variants (for beverages)
Flavor variants (for specific items)
Global add-ons for budget/silog meals
3. Order System (Essential)
Basic cart functionality
Order placement
Simple payment process:
GCash (manual verification)
Cash on pickup
Basic order tracking
Receipt generation (2 letters + 2 numbers)
4. Store Operations (Basic)
Store hours management
Basic order status updates
Simple inventory tracking
